# Sony_Linkbuds-S_Manager
This tool can be used as a simply GUI to flash firmware up or down to Sony Linkbuds S units.
<br>
 It does not care if you have mis-matched fw buds or not. 
 <br>
 You will have to source the firmware: 
 <br>
 https://github.com/lzghzr/MDR_Proxy/tree/master/firmware/2950/MDRID295000
 <br>
 Create a folder called Firmware and place it wherever you put the .exe file.
 <br>
 Drop your fw files from MDR_Proxy into the firmware folder
 <br>Run the exe
 <br>Select your desired fw version from the drop down
 <br>Flash fw...it doesnt care if they are on the same or different fw. 
<br>
 This is used for when you are replacing one bud or another, or simply want to run an old firmware for whatever reason
